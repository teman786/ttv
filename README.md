# 🍽 Restaurant Website Landing Page

A modern and responsive Restaurant/Hotel landing page designed using HTML and CSS. This project focuses on creating an attractive user interface with clean layouts, stylish design elements, and a user-friendly experience for showcasing restaurant services, menus, and booking information.

The website is built to simulate a real-world restaurant or hotel homepage with visually appealing sections and responsive web design principles.

## ✨ Features

* Responsive Website Design
* Modern UI Layout
* Navigation Bar
* Hero Section
* Food/Menu Showcase
* About Section
* Contact Information
* Stylish CSS Animations
* Mobile-Friendly Design

## 🛠 Technologies Used

* HTML5
* CSS3

## 🎯 Purpose

The objective of this project is to improve front-end web development skills by designing a visually appealing and responsive restaurant/hotel website interface using pure HTML and CSS.
